# ML Classification Models Comparison

## Problem Statement

This project implements and compares six different machine learning classification models to predict **_Breaster Cancer_**. The goal is to build an end-to-end ML pipeline that includes model training, evaluation, and deployment through an interactive Streamlit web application.

The assignment requires:
- Implementation of 6 classification models
- Comprehensive evaluation using multiple metrics
- Interactive web application for model comparison and prediction
- Deployment on Streamlit Community Cloud
